# Base de dados — como obter

⚠️ **O arquivo CSV não é versionado neste repositório** (conforme exigência do trabalho). Esta pasta contém apenas as instruções para baixá-lo.

## Origem

**State of Data Brazil 2024–2025** — o maior mapeamento do mercado brasileiro de dados e IA, realizado pela comunidade **Data Hackers** em parceria com a **Bain & Company**. 5.217 respondentes, 400+ colunas.

## Download

1. Acesse: https://www.kaggle.com/datasets/datahackers/state-of-data-brazil-20242025
2. Faça login no Kaggle (conta gratuita) e clique em **Download**.
3. Extraia o `.csv` e coloque-o **nesta pasta** (`dados/`).
4. Confira o nome do arquivo no notebook (variável `CAMINHOS_POSSIVEIS`) e ajuste se necessário.

> Dica para Google Colab: em vez de salvar aqui, use o botão de upload do Colab ou monte o Google Drive e aponte o caminho no notebook.

## Dicionário de dados

O dicionário oficial (descrição de cada coluna `P1_a`, `P2_h`, etc.) está disponível na própria página do dataset no Kaggle, na aba de descrição/arquivos.
